# BookBox-V2
📚 BookBox (Version 2)

BookBox is a web app that lets users search for books in real time using live data. This version improves the earlier prototype by integrating the Google Books API and making the app more dynamic and usable.

# 🚀 What’s New in Version 2

🌐 Integrated Google Books API (real book data instead of static database)

🔍 Real-time search with instant results

⚡ Cleaner and faster UI

🔁 Fallback logic for reliability

# 🚀 Features
Search books by title or author

View book details (title, author, rating, cover)

Dynamic results without page reload

Simple and responsive interface

# 🛠️ Tech Stack
HTML

CSS

JavaScript

Google Books API

# 📂 Project Structure
/project-folder

│── index.html

# 📌 How It Works
Takes user input from the search bar

Sends a request to the Google Books API

Processes the response

Displays results dynamically on the page
